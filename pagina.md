Comandos
-> Abrir a pasta pertinente, clicar com o direito e "open git bash here"
    • Git init -> inicia um rep. Vazio
    • Git add nomeDoArquivo -> coloca o commit em "stage", é como se ele estivesse preparado, esperando para ser empurrado, comitado
    • **git add . -> sobe todos os arquivos novos
    • Git commit -m "Titulo do commit"
    • *Git branch -M "main" -> renomear a branch
    • Git remote add origin link do repositorio -> faz a junção dos repositórios - só é necessário usar uma vez
    • Git push -u origin main
    • Clear - limpa o terminal
    git checkout -b "nova-pagina" -> sai da branch main e entra na branch "nova-pagina"
